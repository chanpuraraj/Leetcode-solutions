Problem 58 - Length of Last word

Given a string s containing words and spaces, return the length of the last word in the string. A word is defined as a contiguous sequence of non-space characters.

Example 1:
Input: s = "Hello World" → Output: 5
Explanation: The last word is "World", which has length 5.

Example 2:
Input: s = " fly me to the moon " → Output: 4
Explanation: The last word is "moon", which has length 4.

Example 3:
Input: s = "luffy is still joyboy" → Output: 6
Explanation: The last word is "joyboy", which has length 6.
