Problem 380: Insert Delete GetRandom O(1)

Design and implement a class RandomizedSet that supports efficient insert, remove, and getRandom operations.

RandomizedSet() – Initializes the data structure.

insert(int val) – Adds the value to the set if it’s not already present; returns true if inserted, otherwise false.

remove(int val) – Deletes the value if it exists in the set; returns true if removed, otherwise false.

getRandom() – Returns a random element from the set, ensuring each element has an equal chance of being selected.

All operations must run in average O(1) time.

Example:
Input:
["RandomizedSet", "insert", "remove", "insert", "getRandom", "remove", "insert", "getRandom"]
[[], [1], [2], [2], [], [1], [2], []]
Output:
[null, true, false, true, 2, true, false, 2]
